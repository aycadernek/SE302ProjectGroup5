Examify'a Hoş Geldiniz!

Bu rehber, sınavlarınızı planlamanıza yardımcı olacaktır.

Yeni Bir Program Oluşturma:
1. Program -> Yeni Program'a gidin.
2. Programınız için bir ad girin.
3. Sınav dönemi için başlangıç ve bitiş tarihlerini seçin.
4. Günlük minimum ve maksimum zaman dilimlerini ayarlayın.
5. Ders ve sınıf listelerinizi (.csv veya .xlsx) içe aktarın.
6. "Program Oluştur" düğmesine tıklayın.

Program Düzenleme:
1. Program -> Programı Düzenle'ye gidin.
2. Değiştirmek istediğiniz programı seçin.
3. Adını ve tarihlerini güncelleyebilirsiniz.
4. Zaman dilimlerini değiştirirseniz, program yeniden oluşturulacaktır.

Program Silme:
1. Program -> Programı Sil'e gidin.
2. Kaldırmak istediğiniz programı seçin.
3. "Sil" düğmesine tıklayın.

Program Dışa Aktarma:
1. Program -> Programı Dışa Aktar'a gidin.
2. Dışa aktarmak istediğiniz programı seçin.
3. Biçimi seçin (PDF, XLSX veya CSV).
4. "Dışa Aktar" düğmesine tıklayın.

Arama:
- Ana ekrandaki arama çubuğunu kullanarak öğrenci kimliği, ders, sınıf veya tarihe göre sınavları bulun.
- Önce bir program seçin, ardından arama teriminizi girin ve kategoriyi seçin.

Dil Değiştirme:
- Dil menüsünden Türkçe ve İngilizce arasında geçiş yapın.

Daha fazla yardım için lütfen destek ile iletişime geçin.
