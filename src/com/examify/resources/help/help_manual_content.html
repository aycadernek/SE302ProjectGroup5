Welcome to Examify!

This guide will help you get started with scheduling your exams.

Creating a New Schedule:

  1. Go to Schedule -> New Schedule.
  2. Enter a name for your schedule.
  3. Select the start and end dates for the exam period.
  4. Set the minimum and maximum daily time slots for exams.
  5. Import your course and classroom lists (.json, .csv or .xlsx).
  6. Click "Generate Schedule".

Editing a Schedule:

  1. Go to Schedule -> Edit Schedule.
  2. Select the schedule you want to change.
  3. You can update the name and dates.
  4. If you change the time slots, the schedule will be regenerated.

Deleting a Schedule:

  1. Go to Schedule -> Delete Schedule.
  2. Select the schedule you want to remove.
  3. Click "Delete".

Exporting a Schedule:

  1. Go to Schedule -> Export Schedule.
  2. Choose the schedule you want to export.
  3. Select the format (PDF, JSON, XLSX, or CSV).
  4. Click "Export".

Searching:

  • Use the search bar on the main screen to find exams by student ID, course, classroom, or date.
  • Select a schedule first, then enter your search term and choose the category.

Changing Language:

  • Go to the Language menu to switch between English and Turkish.
