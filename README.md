# Examify - University Exam Scheduler

## Overview
Examify is a desktop application designed to automatically generate conflict-free university exam schedules. It follows the MVC architectural pattern and is built with Java, JavaFX, and SQLite.

## Features
• Automatic exam schedule generation with constraint satisfaction
• Import data from CSV, JSON, and Excel formats
• Export schedules to PDF, Excel, CSV, and JSON formats
• Search and filter exams by student, course, classroom, or date
• Conflict detection and resolution
• Multi-language support (English, Turkish)
• Local database storage with data integrity
• User-friendly Windows-style interface

## Requirements
• Java 17 or higher
• Maven 3.6 or higher
